# Living Security Lab

A cybersecurity training application and living lab environment designed to simulate real world SOC and security monitoring workflows.

## Project Purpose

This project is being built to strengthen practical cybersecurity skills through hands on work with logs, alerts, SQL investigations, Linux concepts, and incident review.

The long term goal is to evolve this into a living security operations training environment that grows over time with simulated attacks, monitoring, investigations, and security analytics.

## Current Features

- SQLite database named security_lab.db
- Tables for users, devices, login attempts, alerts, and incidents
- Python integration for generating and managing security related data
- Streamlit dashboard for viewing alerts and monitoring activity
- Severity filtering for Low, Medium, High, and Critical alerts
- Functions for alert counts and dynamic security event display
- SQL investigation workflows
- Linux and cybersecurity documentation tied to the lab

## Planned Features
